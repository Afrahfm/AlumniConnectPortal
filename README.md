# Alumni Connect Portal

A full-stack mentoring platform connecting students with alumni for long-term mentorship relationships.

## Tech Stack
- **Backend**: Node.js, Express.js, MongoDB, Socket.io, JWT
- **Frontend**: React.js, Material-UI, Socket.io-client
- **Real-time**: Socket.io for chat and notifications
- **Video Calling**: WebRTC integration
- **Authentication**: JWT with role-based access control

## Features
- Role-based access (Admin, Alumni, Student)
- Alumni profile browsing and filtering
- Mentorship request system
- Real-time chat messaging
- Video calling integration
- Calendar-based meeting scheduling
- Personalized dashboards
- Admin management panel

## Setup Instructions
1. Backend: `cd backend && npm install && npm start`
2. Frontend: `cd frontend && npm install && npm start`
